# PCGEx Pro — FAQ

## Is the free core going away?

No. PCGEx is MIT-licensed and will remain free and open source. The core has 200+ nodes and continues to receive features, fixes, and engine version support. Nothing will be removed or degraded to push people toward Pro.

## Do I need Pro to use PCGEx?

No. The free core is a complete toolkit. Pro adds specialized modules for niche workflows. Most users will never need it.

## Can I ship a commercial game using Pro features?

Yes. Your license is permanent for the version you purchased. You can ship commercial products with no additional fees, royalties, or restrictions beyond FAB's standard EULA.

## What happens if the price goes up after I buy?

Nothing changes for you. You already have your license and continue receiving rolling updates. Price increases only affect new purchases.

## Can I modify the Pro source code?

Yes, for your own projects. FAB's per-seat license allows modification for internal and project use. You can't redistribute the source publicly.

## How does licensing work for teams?

FAB's EULA treats code plugins as per-seat — each person using the plugin in their workflow needs their own license. For studios where that adds up, reach out on [Discord](https://discord.gg/Aze3puAg6T) or [GitHub](https://github.com/Nebukam) and we'll work out volume terms that make sense.

## I'm a Patreon supporter — do I get Pro for free?

Long-standing Patreon members whose cumulative contributions reach approximately $25 (half the Personal tier price) receive a complimentary Pro license. Reach out on Discord when you hit that threshold.

## I've contributed to PCGEx — do I qualify for a free license?

Meaningful contributors may receive a complimentary license. This is handled personally — there's no automated system or formal program. If you've contributed and feel this applies, reach out.

## How do I get GitHub source access after buying on FAB?

Request access to the [PCGEx GitHub org](https://github.com/PCGEx) with proof of purchase. You'll be added to the Pro team, which gives you access to the individual module repos.

## Is there a refund policy?

FAB's standard refund policy applies. PCGEx doesn't maintain a separate refund process.

## Will the price go up over time?

Yes, gradually. As new modules are added, the listing price will increase to reflect the growing scope. But that only affects new buyers — if you already own Pro, you keep receiving rolling updates at no extra cost. Buying earlier gets you more value over time.

## Which engine versions does Pro support?

Active development targets the latest engine version (e.g., 5.7). Bug fixes are backported to the previous version (e.g., 5.6) when feasible, but Epic's PCG framework sometimes changes in ways that make backporting impractical. Older versions aren't updated but continue to work — nothing breaks, you just won't see new features.

PCGEx is a solo project. This support scope is realistic and honest rather than aspirational.

## I'm already using the free core. How do I switch to Pro?

Pro embeds the full free core — it's a superset, not an add-on. Uninstall the free version, install Pro, and you're done. No migration, no redirectors, no project changes. It's seamless.
